Per installarla basta copiare alcuni file nella cartella principale del gioco.

## 📂 Dove trovare la cartella del gioco
Se hai il gioco su **Steam**:
1. Apri la Libreria di Steam
2. Clicca con il tasto destro su **The Last Caretaker**
3. Vai su **Gestisci → Sfoglia file locali**

Si aprirà la cartella principale del gioco (root).

---

## ⚙️ Installazione

Identifica dove hai installato il gioco (la cartella si chiama Voyage).

**apri di nuovo Voyage, Content, Paks e incolla i tre files della mod.**

```text
    📂 Voyage
    └──📂 Voyage
        └── 📂 Content
            └── 📂 Paks
                ├── pakchunk9-Windows_Italiano.pak
                ├── pakchunk9-Windows_Italiano.ucas
                └── pakchunk9-Windows_Italiano.utoc
```

Lancia il gioco per averlo in italiano.

## ⚙️ Disinstallazione

Per rimuovere la mod sarà sufficiente cancellare i tre file

```text
    📂 Voyage
    └──📂 Voyage
        └── 📂 Content
            └── 📂 Paks
                ├── pakchunk9-Windows_Italiano.pak
                ├── pakchunk9-Windows_Italiano.ucas
                └── pakchunk9-Windows_Italiano.utoc
```
